## Folder structure
```plaintext

|
 
├─ backend/

│  │   └─ db.php

│  │   └─ attractieController.php
│  ├─ migrations/        # SQL scripts om database op te zetten

│  ├─ models/            # Klassen of logica voor database tabellen
```
